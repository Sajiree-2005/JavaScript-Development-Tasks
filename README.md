# 📌 JavaScript Development Projects

This repository contains a collection of mini-projects built using **HTML, CSS, and Vanilla JavaScript**. These projects focus on strengthening core JavaScript concepts, DOM manipulation, and building interactive web applications.

---

## 🚀 Completed Projects

I have successfully completed the following 4 tasks:

### 🕒 Digital Clock Web App

- Displays current time in **hh:mm:ss format**
- Updates every second using `setInterval()`
- Clean and responsive UI

---

### 🧮 Calculator App

- Performs basic arithmetic operations:
  - Addition (+)
  - Subtraction (−)
  - Multiplication (×)
  - Division (÷)
- Built using **event listeners and DOM manipulation**
- Interactive user interface

---

### 🌦️ Weather App using API

- Fetches real-time weather data using **OpenWeatherMap API**
- User inputs city name to view:
  - Temperature 🌡️
  - Weather condition ☁️
- Includes error handling for invalid inputs

---

### ❓ Quiz App with Score

- Multiple-choice questions interface
- Tracks user responses
- Displays final score at the end
- Built using arrays, conditionals, and DOM manipulation
